# redis-operator
using redis in SpringBoot , with 5 kinds of data structure.

项目正在开发中，请勿使用

## 简介

Redis一共支持五种数据类型，每一种存储的值及读写的能力如下表所示：

类型     | 存储的值 | 读写能力
:--------:|:---:|:---:
String    | a. 字符串 <br> b. 整数 <br> c. 浮点数 | a. 对整个字符串或者字符串的其中一部分执行操作 <br> b. 对象和浮点数执行自增或自减 
List      | 2   | 3
Set       | 3 | 4
Hash      | 5 | 6
Zset      | 3 | f
